## Descripción general
Proyecto ETL orientado a objetos para procesar archivos SQL (productos, compras y ventas) y generar salidas locales.

## Estructura
- etl/: módulos Python (ETL)
- notebooks/: notebooks para cada ETL (sin resultados)
- data/: archivos SQL fuente
- outputs/: salidas generadas (CSV y SQLite)

## Ejecutar
- Abrir el notebook dentro de notebooks/ y ejecutar celda por celda.
- O ejecutar `python main.py` desde la raíz para procesar los 3 ETL.
